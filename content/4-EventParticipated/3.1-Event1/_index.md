# Report on "Cloud Mastery Series #1: AI From Scratch"

![Cloud Mastery 2026 #1](event1-banner.png)

### Event Objectives

- Explore the real-world applications of Generative AI in project development.
- Introduce the art of communicating with AI through advanced prompt engineering.
- Learn how to build autonomous AI agents and integrate tools into their workflows.
- Showcase practical AI and IoT (AIoT) projects for everyday use cases.

### Speakers

- Anh Dinh Le Hoang (Aiden Dinh) - AI Engineer.
- Vinh Banh Cam - Data Engineer.
- Thinh Nguyen Tuan - DevOps Engineer.

### Key Highlights

#### Advanced Prompt Engineering

- Identified the problems caused by poorly written prompts: wasted tokens, inconsistent outputs, and lower productivity.
- Learned the core parts of a professional prompt: role, instruction, context, input data, output format, few-shot examples, and constraints.
- Explored optimization techniques such as Chain-of-Thought (CoT), Self-Consistency, Tree-of-Thoughts (ToT), and RAG.

#### Building AI Agents with Strands

- Learned how to overcome the limits of traditional LLMs by connecting them to the real world through tools such as APIs and databases.
- Studied the operating flow of Strands Agents: multi-step planning, tool integration, and autonomous decision-making.

#### Practical AIoT Solution - Locker Management System

- Combined hardware components such as Raspberry Pi, Arduino, and sensors with AWS cloud services.
- Applied AWS Rekognition to identify members when borrowing lockers.
- Stored data in S3 and DynamoDB, and handled business logic through Lambda.

### Outcomes

- AI-first thinking: always define the role and context of AI clearly to get the best results instead of relying on generic instructions.
- Agentic architecture: understood the importance of letting AI reason and use tools instead of only generating plain text.
- System integration: saw how IoT hardware and AI cloud services can work together to solve manual operational problems.
- Practical application: made good use of Amazon Q Developer and prompt optimization tools such as Proptimizer to improve coding productivity.

### Application to Work

- Optimize the EduTrust project: apply prompt engineering techniques to build more accurate personalized learning recommendation features.
- Deploy AI agents: experiment with integrating Strands Agents into the system to automate user response workflows.
- Improve coding workflows: use the principles of being clear and specific and using delimiters when working with AI-assisted code generation.

### Event Experience

The workshop provided a hands-on academic experience with real value:

- Learning from experienced engineers: heard directly from AI, Data, and DevOps engineers about how they deploy AI in enterprise environments.
- Lively demo experience: seeing the smart locker system and the Plutus budget management application helped me visualize the data flow between cloud services and devices.
- Exposure to new tools: explored the Proptimizer browser extension, which helps optimize prompts directly in the web interface.

### Lessons Learned

- Prompt engineering is not just about asking questions; it is a structured communication method that saves time and cost.
- AI agents are the future of intelligent applications because they can carry out complex workflows instead of only producing text replies.
- The combination of AI and IoT opens the door to massive automation possibilities for traditional services.

### Conclusion

Cloud Mastery Series #1: AI From Scratch provided a strong foundation for applying Generative AI in real projects. The knowledge and examples from this session are highly useful for building smarter, more practical, and more efficient solutions in EduTrust.
